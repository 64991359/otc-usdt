# otc-usdt
仿币汇（易币付）otc交易所

特色功能
1：自动分单：比如卖家发出1000u的出售，而现在没有刚好要买1000U的买家
系统会自动把卖家的出售订单进行拆分
2：自动归集：
系统会定时自动归集所有用户的usdt到指定的帐号
3:多重验证
用户登陆，或交易时，可以选google身份验证，手机短信验证，邮箱验证等！
