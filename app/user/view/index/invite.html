联盟常见问题答案

您可能会有些问题。我们在下方为您列出了最常见问题的答案。

Paxful是什么？
Paxful是一个点对点市场，让您可以通过300余种支付选项来买卖加密货币。 它的操作有点像eBay，只不过交易对象是数字货币，而且没有限制。 我们的全球用户超过480万。Paxful上的所有交易都由我们的托管服务保驾护航，这意味着您可以安全地进行交易，无需担心资金安全。
从哪里查看自己联盟计划收益？
您可以从自己的联盟计划主页和卖家主页查看收益。
我可以创建用于跟进目的的自定义联盟计划链接吗？
是的，在您的联盟计划主页上，您可以在跟踪 ID 部分创建专属的可跟踪链接，与您的加密货币推荐人共享。
有什么操作禁区？
为确保您了解我们允许和不允许的操作，请参考下方列表，查看作为联盟成员，您需要避免的行为。

  • 创建多个账号 - 在Paxful上您只能拥有一个账号。
  • 从自己的网站导向Paxful网站。
  • 从事欺诈活动以获得联盟成员。
  • 为Paxful、paxful.com. com 或其任何衍生品购买品牌查询的广告，包括错误拼写的版本。
  • 利用其他竞争公司的品牌查询为 Paxful 品牌做广告，并在广告中将Paxful与其他品牌进行比较，让其他品牌处于不利地位。
  • 为了吸引用户发布虚假信息，或以任何形式误导用户。
  • 在Paxful联盟收益之外征收费用。
希望了解更多？请查看我们的联盟计划服务条款。
什么是Paxful联盟计划, 它是如何运作的？
Paxful联盟计划是一个绝佳的机会，让您可以与身边所有人分享Paxful并因此获得收益。方法非常简单——您为Paxful吸引用户，每次他们在Paxful上购买加密货币时，您均可赚取一部分托管费。 对于您向Paxful推荐的用户，您将赚取托管费*的50%， 对于他们向Paxful推荐的用户，您将赚取托管费*的10%.此收益并非一次性的，推荐人每次在Paxful购买加密货币您均可赚取收益。 想要构建自己的加密货币推荐网络十分简单，只需将您的Paxful联盟链接包含在您创作的内容上，通过社交媒体上分享，或在您的网站宣传即可。
该计划的不同层级佣金有何区别？
每当您的联盟成员，或他们的联盟成员在Paxful上成功完成交易，您都会获得支付。 您将收到的金额基于我们的标准托管费用计算，即交易金额的1%，除非托管费用低于1%。如果托管费用低于1%，则根据当前的托管费用计算联盟费用。 支付结构如下： 第1层级：当用户成功购买加密货币，您将收到标准托管费用的50%。 第2层级：当用户成功购买加密货币，您将收到标准托管费用的10%。
什么内容和广告可以吸引更多用户加入我的联盟计划呢？
您可以充分利用很多不同的资源，包括您的博客、您的网站、Google AdWords、Yandex Direct、雅虎/必应广告、移动广告网络、YouTube和其他主流娱乐和金融平台。 社交媒体也是帮助您赚取比特币推荐费的好工具。除此之外，使用追踪 ID 和各种促销活动还可以帮助您发现最佳方式，不断优化您的联盟交易。
联盟计划主页统计数据多久更新一次？
统计数据几乎在您的联盟主页实时更新，因此您在主页上看到的数据应该就是当下的实际数据。
为什么一些推荐人进行了几十笔交易，而另一些仅仅注册了，没有进行任何交易？
用户的需求各有不同。一些用户可能仅仅想在 Paxful 上购买他们的第一枚比特币, 然后一直持有数月数年；另一些人则想用平台买卖更多。联盟成员的交易情况还可能取决于您邀请用户时创建的内容，以及您的目标用户是谁。
我什么时候能拿到收益？
您通过加密货币推荐人赚取的收益通常会在联盟成员完成交易后，立即计入您的联盟钱包。 一旦您的联盟收益累计达到10美元等值比特币，您将可以随时将其转入自己的Paxful钱包。一旦您的收益达到300美元等值比特币，您将需要完成身份验证和地址验证。 提示：您可以在我们的平台上出售您从联盟计划赚得的比特币，以获得更多利润！
在哪里可以找到自己的联盟计划链接？
卖家主页或联盟计划主页都可查看您的专属链接。
在哪里可以查看我的自定义追踪ID？
很快您就可以在自己的主页查看跟踪 ID 了，敬请期待吧！
在寻找推荐人时, 我应该针对哪些国家呢？
Paxful 是全球性平台，不论身处世界何地，只要有互联网就可以访问Paxful。因此，您可以根据自己的需求选择目标市场，无论在本市、本省、本国家还是其他地区都可以轻松实现创收！ 目前，我们的大多数用户来自美国，非洲、亚洲和拉丁美洲是增长速度最快的市场。
Paxful联盟计划的推荐人有没有层级之分，整个项目是如何运作的呢？
您不仅可以通过推荐直接联盟成员获得比特币。Paxful联盟计划分为两个层级，直接推荐人和间接推荐人： 第1层级：直接通过您的链接注册的推荐人。 第2层级：通过您的直接推荐人的链接注册的联盟成员。
如果我的联盟成员与另一个联盟进行交易，我还能获得全额佣金吗？
如果您的两位直接联盟对象互相交易，那么针对该笔交易您将获得25%的整体收益，而不是通常的50%。
我推荐给Paxful的用户没有通过我的链接注册，我还能得到佣金吗？
抱歉，不可以。用户必须通过您的专属联盟链接注册，您才可以获得联盟收益。
我有一个与比特币相关的社交媒体频道，并且用户对买卖比特币比较感兴趣。我应该推荐他们使用Paxful进行交易吗？
当然！Paxful是全球范围内交易比特币最简单的方式之一，您可以随时随地与任何人交易。您的加密货币推荐人可以通过您的联盟计划链接直接注册, 在几分钟内购买或出售比特币！每当他们在Paxful购买，您都会赚取联盟收益。
